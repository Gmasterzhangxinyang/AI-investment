# Reporting Policy

本系统是规则模型生成的投研辅助工具，不构成投资建议或收益承诺。所有客户可见输出必须使用保守、可审计、可复核的话术。

## 必须使用的表达

- `模型触发建仓候选`
- `模型触发平仓提示`
- `进入观察池`
- `不做交易`
- `关注交易`
- `需人工复核`
- `历史诊断显示`
- `数据不足，无法判断`

## 禁止表达

以下词句会被文本安全工具扫描并替换或触发 WARN：

```text
一定上涨
稳赚
无风险
保本
收益承诺
强烈买入
强烈推荐
确定性机会
```

此外，报告应避免直接使用：

```text
建议买入
建议卖出
看好
收益稳定
策略有效
AI认为可以买
```

## LLM 边界

LLM 只能解释证据包，不能：

- 新增标的。
- 改写 `buy_signal/sell_signal/status/rank/score`。
- 把关注池说成建仓候选。
- 承诺收益。
- 绕过数据质量或风控规则。

OpenAI 不可用或超时时，系统必须降级为本地确定性解释，不能阻塞日报生成。

## 固定免责声明

所有 Excel、PDF、dashboard 和问答解释均应保留：

```text
本报告仅为规则模型生成的投研辅助结果，不构成投资建议或收益承诺。
```
