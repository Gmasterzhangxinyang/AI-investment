# 可转债原策略（legacy_v1）说明

原策略是可转债的基础决策层。它先排除价格、强赎、信用、ST、异常 YTM、高溢价率和规模等明显风险，再根据基本面、溢价率、YTM、剩余期限、评级、强赎状态和规模计算基础分。

基础分决定基础等级、候选资格、风险动作和最终排名。系统不会为了凑满 Top10，把弱观察或风险观察标的补进合格候选。

保留原策略是因为它稳定、可复现，并不表示它已经被正式回测证明能够获得绝对收益。结果只用于研究辅助，不是自动交易指令。

- 基础分：基础模型的综合质量分。
- 基础等级：由基础分确定。
- 资格：合格、弱观察、风险观察或排除。
- 未入选原因：明确指出基础分或硬风控原因。
- 排名：只由基础策略决定。
