# 可转债动态辅助（dynamic_v2）说明

当前状态：默认启用在 `legacy_v1` 基础决策层之上。配置名虽然是 `active_strategy=dynamic_v2`，但当前实现仍是辅助 overlay，不是替代基础排名的新总分模型。

动态辅助读取收盘后的四项数据：正股当日涨跌、转债当日涨跌、前一日转股溢价率、当日转股溢价率变化。它只提示短期联动是否值得关注：

- 关注补涨：正股偏强、转债跟涨不足且溢价收缩。
- 谨慎追涨：转债涨幅领先正股且溢价扩大。
- 联动走弱：正股和转债同步明显走弱。
- 数据不足：字段缺失或变化关系无法核验。

动态辅助不改变硬排除、基础分、基础等级、资格、动作和最终排名。正常联动不在主表制造额外提示。

当前代码保持 `score = base_score`；配置里的 0.8/0.2 权重尚未合成为最终排名分，只作为插件配置元数据保留。

这些字段按收盘后数据使用，只用于收盘后复盘和下一交易日参考，不用于当天盘中判断。辅助分尚未经过正式历史验证，不能理解为收益概率或新的买卖评分。
